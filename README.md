# React Native Pin Code

Componente de entrada de código PIN.

## Propriedades

| propriedade | Descrição |
|--|--|
| length | Tamanho do código PIN. **Default: 5**. |
| value | Valor do código. |
| onChange | Handler que retorna o valor digitado pelo usuário. |
